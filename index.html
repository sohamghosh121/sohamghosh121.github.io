Projects

Mining social connections from Flickr YFCC100M datasets

I recently started this project with Dr. Iva to explore if social connections between people can be mined from an images dataset. YFCC100M is a large 100 million size image dataset with many informative tags, which we hope to exploit to build a social graph of the Flickr users who posted the photos and people who are contained in these photos. A possible future application of findings from this project could be in forensics.
Worked with : Dr Iva Bojic, Singapore MIT Alliance For Research and Technology

Final Year Project: Tattoo detection and classification using CNN

This is a niche field that not many researchers have looked into, but is nevertheless important- with the amount of digital media obtained from crime scenes and the importance of tattoos in victim/criminal identification. In this project I am building a classifier for tattoos using the NIST classes to aid in a content based image retrieval application. Part of our work (on tattoo detection) was published in ICB 2016 , where I had the honour of presenting the paper. The paper was also chosen as one of the Best Reviewed Papers
Technologies used : Python, MATLAB, cuda-convnet2
Worked with : Prof. Adams Kong, Biometrics and Foressics Lab (NTU)

Distributed Image Compression in OpenCV

Implemented a distributed image compression algorithm for a real time imaging application based on the idea of compressive sensing. In this, different blocks were encoded at different measurement rates. The Scrambled Block Hadamard Ensemble (SBHE) matrix was used as the measurement matrix, Daubechies 9/7 wavelets as the sparsifying transform, and the SpaRSA algorithm for reconstruction. I tried using multithreading, but it was slower due to extensive shared memory access.
Technologies used : C++, OpenCV
Worked with : Alexander Hanel, Dr. Felix Klanner (Technical University of Munich)

Weblaze: exploring the workings of Webkit GTK browser

This was an exploratory project where I got to learn a lot more about Linux, C++ and JavaScript. This tied in with CS164 Programming Languages and Compilers where I learnt how JavaScript is interpreted in bytecode. I explored the source code of the Webkit GTK browser in order to pinpoint where bytecode translation and interpretation occured, how the JavaScript VM was managed and so on.
Technologies used : C++
Worked with : Kevin Chen, Prof. Dawn Song (U.C. Berkeley)

RHS: A regular expression DSL in Javascript

My team and I realised that composing complex regular expressions are difficult, confusing, and can often be wrong, and are definitely hard to read. What code is really easy to read? The jQuery syntax of chaining method calls. Hence we wanted to create a JavaScript DSL that would allow users to compose regular expressions (and reuse them) in jQuery like syntax. This DSL also allowed for auto generation of a NFA equivalent to the regular expression.
Technologies used : JavaScript, Graphviz

Augmenting Non-Photorealistic Rendering with Image Segmentation and Face Detection

During summer 2014, I worked on improving the performance of an ant-colony based non photorealistic artistic rendering software with image segmentation and face detection in OpenCV. This software took images and turned them into artistic pictures based on parameters: I included facial features and image segment awareness to the ants.
Technologies used : C++, MFC Classes, OpenCV
Worked with : Prof. Ong Yew Soon, NTU

Evaluation Of Smart-Phone Performance For Real-Time Traffic Prediction

I worked on evaluating SVM/SVR for traffic prediction in a real-time setting. I worked on training SVM models with different hyperparameters on a compressed matrix of the traffic network. This was the first time I was exposed to machine learning, and also the first time I was an author for a conference paper.
Technologies used : MATLAB, libSVM
Worked with : Prof. Justin Dauwels, NTU
